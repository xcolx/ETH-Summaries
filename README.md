# ETH Zurich Study Cheat Sheets & Summaries

This repository contains personal cheat sheets, condensed summaries, and exam-oriented notes for various courses taken during my Computer Science studies at ETH Zurich.

All materials are also available online at:  
https://colinrueegg.com/studies

## Disclaimer

These notes are unofficial, may contain errors, and are not affiliated with ETH Zurich or any course staff. They are intended for personal study and revision purposes only.

## Structure

Each course is organized in its own folder.

## Format

- Primarily PDF summaries and cheat sheets  
- Occasionally supporting Markdown or LaTeX sources  

## Purpose

- Fast exam revision  
- High-density concept summaries  
- Supplement to lecture material
